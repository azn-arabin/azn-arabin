## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

To clone this project to your local repo
* npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Arabin1/my_portfolio
   ```
2. Run npm Install
    ```sh
    npm install
    ```
Enjoy.
